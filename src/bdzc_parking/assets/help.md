# 博达智创-管委会停车桥接程序

本程序接收海康威视停车终端上报的过车消息，解析车牌、方向、时间和图片后，按规则转换并发送给大园区停车系统。
博达出口车辆出博达园区、进入上园路，发送 `hobby=in`；博达入口车辆出上园路、进入博达园区，发送 `hobby=out`。

## 使用说明

1. 在 **配置** 中确认监听端口、接收路径和大园区 API 地址。
2. 如需让外部系统访问保存的图片，请配置 `external_url_base`，例如 `https://host/parking-images`。
3. 点击顶部 **HTTP server** 按钮开始或停止接收海康消息；也可在配置中启用自动开启。
4. 左侧查看过车列表，选中条目后，右侧查看详情、图片、接收 HTTP 和发送 API。
5. **模拟发送** 用于手动填写车牌、方向等数据，发送测试给大园区 API。
6. **查看日志** 会用系统默认程序打开当前日志文件。
7. 对默认跳过但已生成 payload 的记录，也可以在右侧详情点击 **手动发送**。
8. 关闭 GUI 时会弹确认框；如果 HTTP server 正在运行，会明确提示退出将停止外部访问和海康上报接收。

## 图片外链说明

- `external_url_base` 为空时，发送给大园区的 `img` 仍然是本地图片文件名。
- `external_url_base` 非空时，`img` 会被拼成完整 URL，例如 `https://host/parking-images/<文件名>`。
- 程序会自动提供对应的图片下载路由，并对图片 GET 做按 IP 限流，防止被恶意刷图。

## 海康威视出入口终端配置

程序的 HTTP server 接收从海康威视出入口终端发送的 HTTP 消息。配置方法如下：

- 登录出入口终端 web 界面
- 参数配置 - 平台接入 - 远程主机，再点击“参数配置”按钮
  - 平台接入方式：http 主机
  - URL：`/park`
  - 填入程序运行主机的 IP 地址和程序端口（默认 `1888`）
  - 按需填入其他选项
  - 点击保存

配置完后，海康主机会在过车信息时，发送消息到 HTTP 服务器。
